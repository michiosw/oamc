# oamc Docs

`oamc` is a local-first LLM wiki for research workflows.

The project separates:

- `raw/` as immutable source input
- `wiki/` as the maintained knowledge layer
- `src/llm_wiki/` as the application that ingests, queries, lints, and serves the wiki

Start here:

- Read the repository overview in the root `README.md`
- Use `docs/architecture.md` for the system shape and runtime model
- Use `CONTRIBUTING.md` if you are changing the product code
