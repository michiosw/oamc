# Contributor Covenant Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in this project a respectful, harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- being respectful in disagreement
- giving actionable, kind feedback
- assuming good intent while staying direct
- focusing on what helps the project and the people using it

Examples of unacceptable behavior:

- harassment, intimidation, or discrimination
- personal attacks or insulting language
- publishing private information without consent
- disruptive behavior that repeatedly derails collaboration

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing these standards.

## Scope

This Code of Conduct applies within project spaces and in public spaces when someone is representing the project.

## Enforcement

Report unacceptable behavior to the maintainer through the private security/contact path described in [SECURITY.md](./SECURITY.md).

All reports will be reviewed and investigated promptly and fairly.
