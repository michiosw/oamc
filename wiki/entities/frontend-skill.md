# Overview
The Frontend Skill is a curated skill package for LLMs (notably [[entities/gpt-5-4]]) designed to generate visually distinctive and deliberate UI designs.

# Roles
- Guides composition, hierarchy, motion, and content structure in generated UIs
- Avoids generic layouts, cards, and weak branding by enforcing a set of hard design rules

# Relationships
- Often installed and used with [[entities/gpt-5-4]]
- Provided by [[entities/openai]] via the skills directory

# Timeline
- Released alongside or after GPT-5.4 in April 2026 for improved UI outcomes
