# Overview
Playwright is an end-to-end browser automation tool commonly used for UI testing and interface validation.

# Roles
- Enables models like [[entities/gpt-5-4]] to inspect, test, and verify frontend work
- Used for multi-viewport testing, behavioral validation, and visual regression

# Relationships
- Interacts with [[entities/gpt-5-4]] via native tool-use capabilities

# Timeline
- First introduced by Microsoft, now widely adopted for web development and LLM-based workflows
