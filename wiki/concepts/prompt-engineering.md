# Definition
Prompt engineering is the process of designing and structuring inputs to language models (LLMs) to steer outputs toward desired, high-quality results.

# Key Ideas
- Constraints and specificity enhance model output
- Narrative and visual context can guide frontend generation
- Rules and skills (like frontend-skill) promote consistency

# Variants
- Hard constraints (explicit rules)
- Soft constraints (style recommendations, mood boards)

# Open Questions
- How much detail should be supplied for optimal creative results?
- How does prompt saturation impact model reasoning and expression?
