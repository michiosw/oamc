# Definition
A design system is a set of standards, tokens, and guidelines that create visual and interaction consistency across a digital product.

# Key Ideas
- Design tokens (typography, color, layout)
- Systemic adherence improves scalability and maintainability

# Variants
- Strict (hard rule) systems
- Flexible (guided) systems

# Open Questions
- How can LLMs reliably interpret and reinforce human design systems?
